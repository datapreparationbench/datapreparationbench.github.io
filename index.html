About

Data Preparation Bench evaluates whether agents can build domain-grounded supervision data from books. The current release focuses on the QA generation side of the project: data is constructed from markdown books, used to fine-tune a base model, and then scored on downstream benchmarks to indirectly reflect data quality.

  • Knowledge sources: domain textbooks and reference books collected from broad-coverage ebook sources
  • Domains: law, medicine, finance
  • Base model: Qwen2.5-7B
  • Instruction-following supplement: Dolly-15k
  • Agent families: Claude, Gemini, GPT-5.2, plus Qwen baseline

Experimental setup

  • Books are converted from PDF to markdown before data construction.
  • Agents generate instruction-response QA pairs under simple or detailed prompts.
  • The resulting supervision is used for fine-tuning, then evaluated on public domain benchmarks.
  • For tasks that rely on string matching, an additional LLM-based review step can be applied to reduce false negatives.

Pipeline

1

Collect books

Gather domain textbooks and long-form reference material with broad topic coverage.

2

Convert to markdown

Transform PDFs into markdown so agents can operate over clean machine-readable text.

3

Generate supervision

Use agents to produce instruction-response QA under different prompting specifications.

4

Fine-tune base model

Train a base model on the generated data, with Dolly added to strengthen instruction following.

5

Measure transfer

Evaluate the fine-tuned model on downstream benchmarks as a proxy for data quality.

Observations

01

Simple prompting is stronger overall

Across Claude, Gemini, and GPT-5.2, the simple prompting variant outperforms the detailed prompting variant on the current public aggregate.

02

Gemini Simple leads the current public slice

Gemini Simple + Dolly ranks first overall and is also the strongest public configuration in medicine and finance.

03

Law shows the highest ceiling

The strongest public law score is higher than the strongest public scores in medicine and finance, suggesting that cross-domain transfer difficulty is not uniform.

04

The baseline remains competitive

Qwen + Dolly remains a meaningful baseline, particularly in medicine and finance, rather than serving as a weak reference point.
